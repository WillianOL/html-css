Texto grande e pequeno

Para deixarmos um texto grande e outro pequeno, usamos a tag "big" e "small" (Tag BIG está obsoleta)

Texto deletado

Para deixarmos um texto "deletado", usamos a tag del.

Texto inserido

Para deixarmos um texto inserido, usamos a tag INS.

Texto sobrescrito

Para deixar o texto como algo do tipo: x20+4, usamos a tag SUP

Texto sobescrito

Para deixar o texto como algo do tipo: H2O, usamos a tag SUB.
